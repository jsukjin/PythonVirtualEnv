블로그 포스터의 슬러그를 생서하세요

본문 : {{ $('Write Content').item.json.choices[0].message.content }}

슬러글나 블로그 포스트의 URL에서 도메인 이름 뒤에 위치하는 부분으로
특정 페이지를 식별하는 역할을 합니다.
슬러그는 보통 해당 포스트의 내용을 요약하는 짧고 설명적인 문구로 구성되며
사용자와 검색 엔진 모두 해당 페이지의 주제를 쉽게 이해할 수 있또록 돕습니다.

예 : www.example.com/intelligent-agents 인경우 intelligent-agents가 슬러그 입니다.
좋은 슬러그는 간결하고 관련 키워드를 포함하여, 불필요한 단어는 피해서 가독성과 SEO를 개선하는것이 중요 합니다.

슬러그는 최대 4~5단어로 구성되어야 하며, 해당 블로그 포스트의 주요 키워드 '{{ $('Get Reserved Data').item.json.fields.primary_keyword }}'를 반드시 포함해야 합니다.

반환 결과는 슬러그 텍트스만 출력해야 됩니다. 
다른 설명이나 텍스트 없이, 슬러그만 출력하여 바로 복사해 블로그 포스트 URL에 붙여넣을 수 있도록 해주세요
슬러그는 반드시 영문으로 사용합니다.
